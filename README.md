# MC3DS-Options-Documentation
- **Documentation for Minecraft 3DS's options.txt, amongst other configuration files.**

## Understanding the Value(s):
```
-1      : Always (force) False
0       : False
1       : True
2       : Always (force) True
[]      : Single List
#[]     : Combo List (multiple)
```
